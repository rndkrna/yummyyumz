# Dashboard YummyYumz

Dashboard operasional statis untuk kebutuhan internal `YummyYumz`. Versi saat ini dipakai sebagai fondasi frontend dan belum terhubung ke backend nyata.

## Isi saat ini

- ringkasan metrik utama
- status operasional
- daftar pesanan terbaru
- catatan tim
- tindakan cepat

Semua angka dan status masih placeholder statis agar antarmuka jujur terhadap kondisi proyek.

## Perintah

- `npm install`
- `npm run dev`
- `npm run test`
- `npm run lint`
- `npm run build`

## Verifikasi terakhir

Status terakhir setelah perombakan dari template Vite:

- `npm run test` lulus
- `npm run lint` lulus
- `npm run build` lulus
